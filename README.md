## WEB前端快速开发结构
## BASE
1. HTML5
2. CSS3
3. JavaScript
4. JS框架：Jquery
5. UI:bootstrap 4、SB ADMIN 2
## START
安装初始环境
```
npm install
```
启动服务
```
npm start
```
## MODEL
1. 文件统一导入：页面只需导入global.js、global.css,其他依赖均在global.js、global.css中导入.
2. 在global.js中，封装了基本的ajax接口.
3. 封装了存取cookie的方法，方便实现登录功能.
